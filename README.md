# Password Manager (Python)

## Overview
A simple password manager application that securely stores user credentials using a master password.

## Features
- Add, search, edit, delete passwords
- Master password authentication
- Encrypted file storage

## Technologies
- Python

## Learning Outcome
Learned secure data handling, file operations, and basic encryption concepts.

## How to Run

1. Clone the repository  
2. Run main.py  
3. Follow on-screen instructions  

## Project Structure

- main.py → application logic  
- Encryption.py → encryption functions  
- vault_gui.py → user interface
